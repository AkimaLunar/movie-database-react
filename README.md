## MyFlix
Deployed to [Surge.sh](http://surge.sh/) at [https://jazzy-bite.surge.sh/]([https://jazzy-bite.surge.sh/])

### Run locally

1. `yarn`
2. `yarn start`
3. _optionally_ `yarn test` to run Jest
